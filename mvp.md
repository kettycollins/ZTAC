# Структура легковагового Zero Trust MVP

Прототип системи розгорнуто у вигляді модульного веб-додатка з такою архітектурою папок:

## 📁 Структура проєкту (Architectural Layout)

Розроблений легковаговий Zero Trust MVP має модульну структуру, де кожен компонент відповідає за свій етап збору телеметрії, автентифікації або контекстного аналізу:

```text
zero-trust-school-mvp/
│
├── app.py                  # Головний веб-контролер (маршрутизація, захист сесій, локалізація)
├── config.py               # Централізована конфігурація середовища та ключів безпеки
├── database.py             # Ініціалізація сховища даних та розгортання початкових таблиць
├── authenticate.py         # Модуль ідентифікації (безпечна SQL-параметризація проти SQLi)
├── policies.py             # Рушій політик (PDP): розрахунок Trust Score та гранульованих прав
├── logging_utils.py        # Модуль аудиту безпеки (SIEM-подібний Clean Text логер подій)
├── requirements.txt        # Перелік залежностей проєкту (Flask тощо)
├── translations.py         # Централізований переклад вердиктів (PDP Reasons)
│
├── data/
│   └── users.db            # Локальна зашифрована база даних облікових записів та їхніх ролей
│
├── logs/
│   └── access_logs.json    # Журнал реєстрації подій безпеки у строгому текстовому форматі JSON
│
├── static/
│   └── style.css           # Таблиці стилів адаптивного інтерфейсу (світла/темна теми)
│
└── templates/              # Візуальні шаблони та інтерфейси системи
    ├── index.html          # Головна стартова сторінка-вітрина системи
    ├── login.html          # Форма входу з Context-Aware UI (інтерактивний блокувальник VPN)
    ├── denied.html         # Екран вердикту PDP (ALLOW-проміжний / DENY-екран жорсткого блокування)
    ├── resource_page.html  # Карта об'єктів інфраструктури з гранульованими правами на картках
    ├── admin_dashboard.html# Панель моніторингу безпеки адміністратора (SIEM-інтерфейс логів)
    ├── notice_guest.html   # Сторінка-заглушка публічного гостьового контуру (без картинок)
    ├── notice_student.html # Сторінка-заглушка захищеного студентського порталу
    └── notice_teacher.html # Сторінка-заглушка внутрішнього викладацького кабінету
```
